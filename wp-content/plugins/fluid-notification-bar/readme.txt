== Description ==

<blockquote>
Featured on <strong>WPdean, WPjournals, Codegeekz, Designcoral and icegram</strong> blogs, thank you for writing about us.
</blockquote>

Fluid notification bar plugin adds the cool responsive notification bar with <strong>call-to-action</strong> button on the top of your website. This notification bar uses catchy animation for the text to grab visitors attention. It can be used to-

<ul>
<li> Welcome your blog/website visitor </li>
<li> Announce the giveaways for your website or blog </li>
<li> Link to your blog post or particular page in your website </li>
<li> Say an important message such as server downtime or website maintenance alerts </li>
<li> <strong>Now you can add Social Media buttons</strong> </li>
</ul>

<strong>New Features of Fluid Notification Bar</strong>
<ul>
<li>Customizable animation effect for text</li>
<li>Action button to navigate to a link</li>
<li>Social media buttons (Google+, Facebook, Twitter, Pinterest & Linkedin)</li>
<li>Customizable font size, font color as well as background color</li>
<li>You can set interval for notification bar</li>
<li>You can set delay for notification bar</li>
<li>Works well with Bootstrap's fixed navigation (support for navbar-fixed-top)</li>
</ul>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
